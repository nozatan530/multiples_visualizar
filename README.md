# Multiples Visualizer

1〜20 の倍数をグリッド上で視覚的に確認できるツールです。

## 機能

- **倍数の表示** — 1〜20 の数字を選ぶと、グリッド上の倍数が色で光る
- **複数選択** — 複数の数字を選ぶと共通の倍数（最小公倍数）が紫で光る
- **LCM 表示** — 選択した数字の最小公倍数（LCM）をリアルタイムで表示
- **アニメーション** — 1〜20 の倍数を順番に自動で点灯
- **グリッドサイズ変更** — 100〜1000 まで表示範囲を調整可能
- **ダーク / ライトテーマ** 切り替え対応

## 使い方

1. 数字ボタン（1〜20）をクリックして倍数を表示
2. 複数選択すると共通倍数が紫で強調される
3. `▶ 順に点灯` で 1〜20 のアニメーションを再生
4. `✕ リセット` で選択をクリア

## デモ

`index.html` をブラウザで開くだけで動作します。インストール不要。
